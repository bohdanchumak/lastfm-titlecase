# Changelog

## 1.1.0

> February 06, 2026

### Added
- Configuration page for customizing word lists (lowercase, uppercase, capitalized)
- Capitalize words after dots

### Changed
- Non-English titles are now fully sentence-cased (first letter uppercase, rest lowercase) instead of preserving original casing
- Word override matching is now Unicode-aware

## 1.0.0

> January 28, 2026

### Added
- Automatic title case correction for song, album, and artist names on last.fm
- Sentence case handling for non-English titles
- Dynamic content support
